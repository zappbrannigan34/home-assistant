# CO2 Ventilation Control — Drivent V2 (zap_left)

Автоматическое управление проветриванием через оконный привод Drivent V2 на основе уровня CO2.

---

## Почему не PID?

PID-регулятор дёргает привод на **каждое измерение**: CO2 постоянно меняется из-за внешних условий, и система никогда не приходит в равновесие. Результат — привод работает каждую минуту, изнашивается за год.

Этот пакет использует **гибридное зонное управление по длинному тренду**:
- Смотрим **30-минутное сглаженное среднее** CO2 и **30-минутное изменение**
- Двигаем привод **только когда длинное окно подтверждает необходимость**
- Шагами по **5% или 2%** (не микродвижения по 1%)
- Обычная оценка и регулировка — **раз в 30 минут**

Реальных движений привода: **20-40 в день** вместо 1440 при PID.

## Алгоритм

### CO2 Trend Calculator

Пакет использует два статистических сенсора:
- `sensor.ventilation_co2_mean_30m` — сглаженное среднее CO2 за 30 минут
- `sensor.ventilation_co2_change_30m` — изменение CO2 за 30 минут

Тренд определяется по `sensor.ventilation_co2_change_30m`:

| Разница | Тренд |
|---------|-------|
| > +30 ppm | `rising` |
| < -30 ppm | `falling` |
| в пределах ±30 | `stable` |

### Зоны CO2 (target по умолчанию 650 ppm)

| Зона | Сглаженный CO2 (при target=650) | Rising | Stable | Falling |
|------|--------------------------------|--------|--------|---------|
| Сильное превышение | >800 | +5% сразу | +5% сразу | стоим |
| Умеренное превышение | 700-800 | +2% после 2 окон | +2% после 2 окон | стоим |
| Околоцелевая (deadband) | 600-700 | +2% после 2 окон | стоим | стоим |
| Ниже цели | 500-600 | стоим | стоим | -5% после 2 окон |
| Значительно ниже | <500 | стоим | -5% сразу | -5% сразу |

### Подтверждение сигнала

В умеренных зонах пакет требует **2 одинаковых 30-минутных сигнала подряд**, прежде чем двигать привод. Это убирает дёрганость на нестабильных воздушных потоках.

### Rate Limit

- Оценка позиции: каждые 30 минут
- Минимальный интервал между реальными движениями: 30 минут
- Cooldown хранится явно в `input_datetime.ventilation_last_adjustment`

## Безопасность

| Событие | Действие |
|---------|---------|
| T улицы < min порога | Закрыть окно + уведомление |
| Перегрузка привода | Закрыть окно + уведомление |
| CO2 сенсор unavailable (>2 мин) | Закрыть окно + уведомление |

## Состав пакета

### Template Sensors

| Sensor | Описание |
|--------|----------|
| `sensor.co2_trend` | Тренд CO2: rising / falling / stable |
| `sensor.ventilation_co2_mean_30m` | Сглаженный CO2 за 30 минут |
| `sensor.ventilation_co2_change_30m` | Изменение CO2 за 30 минут |
| `sensor.ventilation_recommended_position` | Рекомендуемая позиция окна (0-100%) |
| `sensor.ventilation_co2_error` | Отклонение CO2 от цели (ppm) |

### Input Helpers

| Entity | Описание | Default |
|--------|----------|---------|
| `input_number.ventilation_target_co2` | Целевой CO2 | 650 ppm |
| `input_number.ventilation_min_position` | Минимальная позиция | 0% |
| `input_number.ventilation_max_position` | Максимальная позиция | 80% |
| `input_number.ventilation_min_outdoor_temp` | Мин. уличная температура | -10°C |
| `input_datetime.ventilation_last_adjustment` | Последняя реальная регулировка | 2000-01-01 00:00:00 |

### Automations

| Automation | Описание |
|------------|----------|
| `ventilation_control_main` | Основное управление позицией |
| `ventilation_safety_close` | Аварийное закрытие |
| `ventilation_device_error` | Уведомления об ошибках привода |

### Используемые внешние entity

| Entity | Тип | Назначение |
|--------|-----|-----------|
| `sensor.sensor_zap_co2` | Source | CO2 в комнате (ppm) |
| `cover.drivent_zap_left_driventokonnyi_privod` | Device | Привод окна (0-100%) |
| `binary_sensor.drivent_zap_left_datchik_peregruzki` | Device | Датчик перегрузки |
| `weather.forecast_home_assistant` | Source | Уличная температура |

## Установка

1. Скопируйте `packages/ventilation/` в `/config/packages/`.
2. **Перезагрузите Home Assistant** (полный рестарт — reload не создаст input_number и trigger-based sensors).
3. Проверьте, что `sensor.sensor_zap_co2` и `cover.drivent_zap_left_driventokonnyi_privod` доступны.
4. Настройте параметры через UI:
   - Целевой CO2 (по умолчанию 650 ppm)
   - Минимальная уличная температура (по умолчанию -10°C)
   - Максимальная позиция окна (по умолчанию 80%)
